ping-pump

ใช้โปรแกรมนี้ในการ กระทุ้งให้ pump-server มันตื่นอยู่เสมอ ไม่งั้นมันหลับ เมื่อเปิดเครื่องทุกครั้ง ให้ run โปรแกรมนี้ มันจะได้คอยไปกระทุ้ง server (ภาษา dev คือให้มันตีปิงปองกัน)

Download & Install

Download โปรแกรม ping-pump แล้วเก็บไว้ใน folder นึง แล้วคอย run มันทุกครั้งหลังจากเปิดเครื่อง

ขั้นตอน

  1. Download
  2. เก็บไฟล์ไว้ใน folder เช่น \pump
  3. Unzip จะได้ไฟล์ ping-pump.bat
  4. การ run ก็ double-click ไฟล์นี้ มันจะเปิด terminal window ขึ้นมา แล้วมีข้อความต่างๆ แสดงขึ้นมา ถ้ามัน ping ไปแล้ว มีข้อความตอบกลับว่า 200 แสดงว่า server ตื่นอยู่ เรียบร้อย ก็ปล่อยให้มัน run ไปเรื่อยๆ ตลอด 24 ชั่วโมงเลยก็ได้ หรือจะเอาเฉพาะตอนเปิดเครื่อง desktop ก็ได้

โปรแกรมนี้มันจะตีปิงปองทุก 10 นาที เพราะฉะนั้น ไม่ต้องปิดมัน ปล่อยมัน run ของมันไป เวลาปิดเครื่องก็ปิดไปเลยได้ พอเปิดเครื่องใหม่ ก็มา run มันใหม่
