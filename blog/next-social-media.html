Next Social Media

ทุกวันนี้ social media ตัวเทคโนโลยีเองน่าจะเรียกว่า เป็น version 1.0 ก็อาจเรียกได้ เพราะมันเป็นการทำงานเบื้องต้น ให้คนแชร์ข้อมูลสู่สาธารณะ ได้ง่ายๆ คือทุกคนสามารถ post เรื่องราวต่างๆได้ แล้ว comment กันได้ แต่สิ่งที่เรากำลังคิดต่อไป ก็คือ social media ยุคหน้า จะต้องเป็นยังไง

เบื้องต้นเลย เจ้า next social media นี่จะต้องเข้ามาทดแทน การสื่อสารของโลก ทั้งหมด นั่นหมายความว่าอะไร ก็คือพวกโทรศัพท์ อีเมล SMS chat อะไรต่างๆนี่ จะต้องถูกกลืนด้วย nex-social นี้ทั้งหมด ทุกคนต้องสามารถหยิบมือถือขึ้นมาแล้วใช้เจ้า nex-social นี้ กับใครก็ได้ คือสื่อสารไปได้หมด เช่นเราต้องการส่งข้อความถึงประธานาธิบดี ก็อาจส่งไปได้เลย เช่น
Hello @president หรือจะ
Hello, @world
อะไรแบบนี้ ง่ายๆ จะส่งเงินก็
send 1000 $xyz to @john
ได้เลย (สมมุติ $xyz คือสกุลเงินดิจิตอล)

ทีนี้เราจะมาลอง list กันดูว่าสิ่งที่เราต้องการใน nex-social ควรมีอะไรบ้าง

  1. สามารถใช้แทนการสื่อสารทุกชนิดในโลกได้
  2. เจ้าของข้อความเป็นเจ้าของอย่างแท้จริง
  3. แน่นอนว่าข้อมูลของ user จะต้องถูกเข้ารหัส และมีแต่ user เท่านั้นที่สามารถเห็นได้ทั้งหมด ซึ่งพวก operator หรือบริษัทเจ้าของ platform จะต้องไม่สามารถเห็นได้ เรื่องนี้ ให้ดี ควรมีกฎหมายที่ใช้ควบคุมด้วย
  4. สื่อสารกันได้ทั้งหมด ไม่ว่าเราจะสื่อสารหาใคร ยกตัวอย่างทุกวันนี้เราจะใช้ x.com ส่งข้อความหาคนใน ig ตรงๆ ไม่ได้ แต่ถ้าเป็น nex-social ต้องทำได้ทั้งหมด คือทุก app ต้องสื่อสารกันได้หมด เหมือนกับที่เราใช้โปรแกรมอีเมลยี่ห้อ A ส่งอีเมลหาคนที่ใช้ยี่ห้อ B มันก็รับส่งข้อมูลกันได้ โดยไม่จำเป็นต้องเป็นยี่ห้อเดียวกัน ต่าง platform กันก็ทำได้
  5. ทุกคนในโลกสามารถเข้าถึง internet/digital และเข้าถึง nex-social ได้เท่าเทียมกัน คือคุณมีมือถือ + net คุณก็มี nex-social ได้ แล้วจะสื่อสารหาใคร จะ post แบบ public ก็ย่อมทำได้ทันที
  6. กิจกรรม ธุรกรรม อะไรของมนุษย์ ก็ควรจะสามารถส่งไปในข้อความของ nex-social ได้อย่างง่ายๆ เหมือนที่เรายกตัวอย่างไปแล้ว ข้างบน ไม่ว่าการระบุถึงคนรับสาร การจะส่งเงินไปให้ใคร อะไรต่างๆเป็นต้น

ประมาณนี้ก่อน คร่าวๆ เพื่อให้พอนึกภาพออก สิ่งที่เรากำลังพูดกัน เรียกว่าเป็น idea ก็แล้วกัน วันนี้เราอาจยังไม่เห็น แต่ว่า การพัฒนาการของการสื่อสารของมนุษย์ มันควรจะพัฒนาไปอย่างไร อย่างทุกวันนี้ ก็ต้องยอมรับว่า social media กลายเป็นช่องทางสื่อสาร หลัก ของมนุษยชาติไปแล้ว พวก web site, email, sms, โทรศัพท์ จดหมาย อะไรพวกนั้น กลายเป็นเรื่องโบราณ ล้าสมัย ไปหมดแล้ว แม้จะมีการใช้อยู่ แต่ว่า มันเป็นช่วงเวลาแห่งการเปลี่ยนผ่าน (transition period) จนในที่สุดพวกนั้นก็จะหายไป เหมือนทุกวันนี้ไม่มีใครถ่ายรูปแล้วต้องเอาฟิลม์ไปล้างที่ร้านอัดรูปอีกต่อไป
