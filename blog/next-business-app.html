#next_business_app

ทุกวันนี้ business app ยังเป็นอะไรที่ราคาแพง องค์กรใหญ่ๆ หรืออย่างน้อยก็ต้องลงทุนหลายๆล้าน อันนี้ระบบธุรกิจแบบทั่วไป บางโปรเจค โดยเฉพาะของรัฐ ได้ยินว่าจ้างกันหลักเป็นพันๆล้าน หรือหมื่นล้าน ก็มี ไม่รู้แพงอะไรกันขนาดนั้น ซอฟท์แวร์ธุรกิจระดับโลกมีหลายตัว ในนี้ไม่อยากกล่าวถึง สมัยก่อนก็ยุ่งเกี่ยวอยู่เหมือนกัน ขายกันเป็นจำนวน user และคิดต่อโมดุลอีก รวมๆแล้ว ค่า license + ค่าบริการติดตั้ง-setup-ปรับแต่ง + ค่า maintenance อีก เรียกว่า ลูกค้าบริษัทเล็กๆ ก็หน้ามืดไปเลยทีเดียว

พอต่อมามียุค open source เกิดขึ้น คือพวก developer ทั้งหลายช่วยกันเขียนขึ้นมา แล้วเปิด source code เปิดให้ใครก็เอาไปใช้ได้ อะไรพวกนี้ ปรากฎว่า มีบริษัท-องค์กรต่างๆ ทั่วโลก ได้ใช้ประโยชน์มากพอสมควร คือลดรายจ่ายค่า license & maintenance ที่ถูกผูกขาดโดยบริษัทซอฟท์แวร์ระดับโลกพวกนี้ไปได้มากมาย อย่างทุกวันนี้ แม้ทีม @nex-era เองก็ยังทำ open source บางตัวให้กับลูกค้าอยู่ เช่น odoo ก็ถือว่าใช้ได้

ทีนี้เรากำลังจะคิดถึง business app ในยุคถัดไป ว่าควรจะเป็นยังไง โดยจะลอง list ให้ดูว่าเราอยากจะเห็นอะไรบ้าง

  1. ทุกคนต้องสามารถใช้ได้ ไม่ว่าจะเป็นธุรกิจแบบคนๆเดียว หรือเป็นหมื่นเป็นแสนคน หรือเป็นแบบกลุ่มธุรกิจที่มีบริษัทในกลุ่มเป็นพันๆบริษัท ก็ต้องทำได้ ใช้งานได้เหมือนกัน ไม่จำเป็นต้อง install & setup แต่อย่างใด คือพอ sign-up แล้วก็เริ่มใช้งานได้เลย
  2. เป็นแบบ web คือ user มีแค่ browser กับ internet ก็ใช้งานได้ทันที
  3. รองรับกระบวนการทำงานในธุรกิจทุกประเภท รวมถึงใช้ได้กับระบบภาษี และเงื่อนไขต่างๆ เฉพาะของแต่ละประเทศ
  4. integrate กับพวก social media เพื่อการสื่อสารกับคนที่เกี่ยวข้อง หรือใครก็ตาม ได้ทันที ยกตัวอย่างเช่น สามารถส่ง quotation ไปให้ลูกค้า ผ่าน social media จากนั้นลูกค้าก็ confirm ใบเสนอราคานั้นได้ทันที ดังนั้นการทำงานทั้งหมดก็ทำผ่าน online/internet ได้ทั้งหมด
  5. เป็น mobile-first คือหน้าจอออกแบบมาเพื่อมือถือ tablet โดยเฉพาะ แต่ก็ใช้งานกับจอประเภท laptop/notebook ได้
  6. clean, easy, and inteligence.
  7. internationalized / รองรับทุกภาษา

ประมาณนี้ก่อน คร่าวๆ อาจจะมีต่อเติมแก้ไขไปเรื่อยๆ อันนี้ให้มองเห็นภาพ ดังนั้น ความก้าวหน้าของระบบธุรกิจ จะต้องไป serve ให้กับทุกๆคน ทำให้ธุรกิจน้อยใหญ่ สามารถมีระบบดีๆได้เหมือนกัน ตรงนี้ก็จะไปช่วยเสริมสร้างศักยภาพ ให้ทุกคน สามารถมีข้อมูล ระบบการทำงานทางธุรกิจ ที่ไม่แพ้บริษัทใหญ่ จะได้เอาฝีมือล้วนๆ มาแข่งกัน ไม่เอาเปรียบกันทาง technology อีกต่อไป จบตอนที่ 1 ไว้เท่านี้ก่อน เจอกันตอนต่อไป
