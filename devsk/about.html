dev-skill เป็นโปรแกรมที่เก็บ dev skill profile ไว้ เพื่อให้ทุกคนเข้ามาค้นดูได้ ไม่มีข้อมูลส่วนบุคคล แต่เน้นที่ dev skill เป็นหลัก ดังนั้นทุกคนสามารถมาป้อน skill ของตัวเองไว้ ทีนี้เมื่อมีฐานข้อมูลของ skill แล้ว ใครอยากจะจ้าง จะหางาน จะหาคนทำโปรเจค จะลงทุนในโปรเจคซอฟท์แวร์ ก็เป็นเรื่องง่าย

เบื้องต้น ก็คงจะต้องชวนกันมาสร้าง dev-skill profile กันเยอะๆ แล้วค่อยพัฒนาโปรแกรมนี้ไปเรื่อยๆ โปรเจค dev-skill นี้เป็นลักษณะแบบ open source, open team คือใครอยากช่วยทำ ก็เชิญมาช่วยกันได้ ขณะนี้เริ่มต้นจากคนในกลุ่ม dev-vold ใน facebook 3-4 คน ถ้าโปรเจคนี้ไปได้ดี ก็จะพัฒนาต่อไปเรื่อยๆ อย่างน้อยก็อาจมีส่วนช่วยสร้างศักยภาพงาน dev ในประเทศไทย
