Software

เราทำงานด้านซอฟท์แวร์ มีทีมงานจำนวนหนึ่ง และหากงานใหญ่ๆ ก็สามารถ scale team ให้พอเหมาะกับขนาดงานได้ โดยมากเราทำพวกนี้

  • ออกแบบ / design
  • พัฒนา / dev
  • ดูแลทางเทคนิค / tech support
  • นำซอฟท์แวร์พวก open source มาช่วยธุรกิจลูกค้า
  • web app / mobile app

เรารับออกแบบ-และพัฒนาซอฟท์แวร์ ให้กับลูกค้าทั่วไป มุ่งเน้นที่งานขององค์กร เช่นพวกระบบธุรกิจ ต่างๆ เช่น sales, purchase, inventory, warehouse, stock, product, customer, accounting, finance, manufacturing, production, HR, dashboard เป็นต้น

งานส่วนใหญ่เป็นงานสำหรับพวกธุรกิจ หรือองค์กร เพราะเราก็มีความรู้ความเข้าใจเรื่องธุรกิจ-องค์กรอยู่แล้ว จึงทำให้เราสามารถออกแบบซอฟท์แวร์เพื่อมาตอบสนองธุรกิจของลูกค้า ได้เป็นอย่างดี คุยกันเข้าใจง่าย พร้อมเสนอแนะทางออกใหม่ๆ ร่วมกัน ไม่ใช่ว่าจะก้มหน้าก้มตาทำทางเทคนิคอย่างเดียว ไม่เข้าใจลูกค้าเลย ไม่ใช่อย่างนั้น

นอกจากการออกแบบขึ้นมาใหม่ๆสดๆเลยแล้ว เรายังช่วยให้ลูกค้าใช้งานซอฟท์แวร์พวก open source เช่นพวก odoo, alfresco และอื่นๆอีกหลายตัว เพื่อให้ลูกค้าได้ใช้งานซอฟท์แวร์สำเร็จรูป ที่ไม่จำเป็นต้องทำขึ้นมาใหม่จาก 0 ก็จะทำให้ได้ระบบใหม่เร็วขึ้น ส่วนมากเราทำพวกซอฟท์แวร์เกี่ยวกับธุรกิจ-องค์กร

รายการ technology ที่เราใช้ เบื้องต้น ประมาณนี้

  • java script, node.js, java, c#, python, flatter
  • mongodb, mysql, postgres, ms sql
  • aws cloud, alibaba cloud, azure cloud
  • linux, windows, ubuntu, mac
